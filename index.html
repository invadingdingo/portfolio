MY WORK

Bird Brain

Bird Brain is a puzzle game that brings new life to traditional top-down puzzle mechanics, like levers and pushable rocks. Players are able to switch between first person and top-down perspective to navigate levels. Each perspective causes certain elements to behave differently. For example, in top-down 2D, height does not exist, so a raised ledge in first person can simply be walked onto in top-down.
  • Unity
  • C#
  • Krita

beatZONE

beatZONE is a rhythm-based bullet hell game where the boss's attacks sync perfectly with the music. Inspired by the theme of "limited," we designed it so the player can only shoot while within a specific zone, adding a unique layer of challenge. Developed in just one week for Bullet Hell Jam 2023, it achieved a notable 29th place overall.
  • Unity
  • C#
  • Ableton

RIBBIT - Choice Based Dialogue

I’m part of a team developing Ribbit, a game that features a dialogue system built on a node/branch structure, allowing player choices to dictate the narrative flow by determining the next visited node. While the game is still in development, you can check out the demo linked below to see the dialogue system in action, along with several other features I contributed to.

Difference

Difference is a creative twist on the classic spot-the-difference genre. Inspired by Inscryption, I set out to replicate the 3D grid-based movement from that game. While the project remains unfinished, the movement mechanic was implemented flawlessly. Although development slowed when I started a new job, I'm proud of the progress I made, especially solving the challenges around the movement system.
