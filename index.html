MY WORK

beatZONE

beatZONE is a (successful!) attempt at something rhythm based. I loved the idea of a boss fight where the bosses attacks are timed to the music in the game, so taking that idea, and the theme for the jam (limited), me and a friend developed beatZONE. We implemented the theme by only allowing the player to shoot while in a designated zone. beatZONE was developed for Bullet Hell Jam 2023 in a week, scoring 29th place overall.
  • Unity
  • C#
  • Ableton

RIBBIT - Choice Based Dialogue

This is for a game I've been working on with a team called Ribbit. The dialogue system uses a node/branch structure to navigate between dialogue nodes. Choices determine the next visited node. The game is totally unfinished, but there is a demo linked below where you can see the dialogue system (and a few other things I developed) in action!
  • Godot
  • GDScript

Difference

Difference is a fun take on the classic spot-the-difference genre. I developed it shortly after playing Inscryption, and really wanted to recreate the movement in that game (3D grid-based). The game is largely unfinished, but the movement mechanic works flawlessly! Unfortunately, I began developing this right around when I started working a new job, so it fell to the wayside. That said, I am very proud of the progress I made on it and solved a fun problem while making it.
  • Unity
  • C#
  • A LOT of 3D Math
